# 🍣 СушиТуса

**Демо:** [Смотреть сайт](https://sushi-tusa65.tilda.ws/)

---

## 📌 Описание

Сайт-визитка для доставки суши «СушиТуса», выполненный на платформе Tilda.

Проект включает 3 страницы, собранные по макету с адаптивом под мобильные устройства.

---

## 🛠 Технологии

- **Tilda** (Zero Block, конструктор сайтов)
- **HTML / CSS** (стилизация, кастомизация)
- **JavaScript** (слайдеры, табы, модальные окна)
- **Voron.dev** (сторонняя библиотека для слайдеров и табов)
- **Яндекс.Карты** (iframe-конструктор)
- **Tilda CRM** (форма отправки заказа)

---

## ✨ Особенности

- Встроенная корзина с отправкой через Tilda CRM
- Всплывающие попапы с информацией о товаре
- Слайдеры и табы на JS (Voron.dev)
- Перелинковка между страницами
- Плавный скролл
- Ховеры на кнопках и карточках
- Header и Footer подключены как отдельные страницы и автоматически дублируются на всех страницах
- Адаптив

---

## 🔧 Структура

- **Главная страница** — приветствие, акции, популярные позиции
- **Меню** — список товаров с фильтрацией по категориям
- **Акции** — текущие предложения и скидки

---

## 📱 Адаптивность

Проект корректно отображается на:
- Десктоп
- Мобильные телефоны
